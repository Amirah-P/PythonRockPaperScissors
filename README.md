<h1>Rock, Paper, Scissors Game: Randomisation and Python lists</h1>

<h2>Description</h2>
This project was created using randomisation and python lists on PyCharm. The user is prompted to choose 0, 1 or 2 and the computer chooses using the random module. The three possible hand signs are stored as ASCII art inside a list. Once the choices are decided the program uses a series of if/elif/else statements to determine whether the user wins, loses, or ties. The project reinforces Python fundamentals like indexing, importing modules, and implementing decision-making logic while creating a simple, interactive game.

<h2>Languages and Utilities Used</h2>

- <b>Python (PyCharm)</b> 
- <b>ASCII ART</b>
- <b>Python 3.12.4 Documentation</b>
